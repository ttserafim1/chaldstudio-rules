ПРАВИЛА CHALD STUDIO

Актуальная версия от 01.07.2023

1.1

Уважительное общение

Все участники команды обязаны поддерживать дружелюбную атмосферу. Запрещены оскорбления, унижения, провокации и любые формы дискриминации.

Наказание: Предупреждение → Варн → Блокировка чата на 3 дня → Исключение
1.2

Конфиденциальность

Запрещено распространять любую внутреннюю информацию команды без разрешения руководства.

Наказание: Варн → Блокировка доступа → Исключение
2.1

Работа с проектами

Запрещено вносить изменения в проекты без согласования с ответственным за проект.

Наказание: Варн → Блокировка доступа на 1 неделю → Исключение
2.2

Использование читов

Строго запрещено использование любых читов, эксплоитов или нечестных преимуществ в проектах команды.

Наказание: Блокировка на 1 неделю → Исключение
2.3

Оскорбление администрации

Запрещено оскорблять, угрожать или проявлять неуважение к администрации команды.

Наказание: Варн → Блокировка на 3 дня → Исключение
3.1

Активность

Участники команды должны поддерживать минимальный уровень активности. Длительное отсутствие без предупреждения может привести к исключению.

Наказание: Предупреждение → Исключение

Дополнительная информация

  • 1 варн = 1 предупреждение. 3 варна = временная блокировка
  • Особо грубые нарушения могут караться немедленным исключением
  • Администрация оставляет за собой право изменять наказания по ситуации
